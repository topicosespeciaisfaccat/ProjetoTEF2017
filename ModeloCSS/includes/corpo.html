Quem somos:

O Sistema Indica Bônus é voltado para caixas de abastecedoras de combustiveis. Fazemos todos o controle do caixa, desde o controle de vendas até calculos de promoções.

Promoção Indica Bonus

O sistema de bonificações deverá funcionar basicamente através das indicações realizadas por clientes, onde cada cliente ao indicar um outro irá ganhar um bônus (em R$) para cada litro de gasolina abastecido. Exemplo:
1 - Pedro se torna cliente do posto de gasolina;
2- Pedro indica Joana para abastecer (Pedro ganha R$ 0,02 para cada litro de gasolina abastecido pela Joana);
3- Joana indica Saulo para abastecer. Cada vez que Saulo abastecer, Joana ganha R$ 0,02 para cada litro de gasolina abastecido e Pedro também ganha mais R$ 0,02;
4- E assim sucessivamente, até atingir o nível máximo de dois indicados diretos e mais dois indiretos;
